# 纸牌游戏 (Solitaire)

一款经典纸牌游戏，支持 Android 手机安装。

## 功能

- 7列11行经典发牌
- 必须同花色才能移动
- 非顶头优先的空列规则
- 自动判断胜利/死局
- 反悔、重新开始、新局游戏
- 手机触控优化

## 构建 APK

项目使用 GitHub Actions 自动构建，推送到 main 分支即可触发。

### 手动构建

1. 安装 [Android Studio](https://developer.android.com/studio)
2. 打开本项目
3. Build → Build APK(s)

## 下载

前往 [Actions](../../actions) 页面下载最新构建的 APK 文件。
